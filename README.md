# World Cup Predictor

A football analytics and prediction platform built using machine learning and data science techniques.

## Features

* Predicts match outcomes (Win / Draw / Loss probabilities) between two national teams using **Random Forest**
* Predicts expected scorelines using **Poisson Regression**
* Supports both **neutral** and **non-neutral venue** predictions
* Clusters football players into different playstyle categories using **K-Means Clustering**
* Evaluates player market value based on:

  * Goals
  * Goal Contributions (G/A)
  * Minutes Played

## Tech Stack

* Python
* Pandas
* Scikit-learn
* XGBoost
* Flask
* HTML/CSS/JavaScript

## Machine Learning Models Used

* Random Forest
* Poisson Regressor
* K-Means Clustering

## Project Goals

This project aims to combine football analytics, machine learning, and interactive visualizations to create a modern World Cup prediction platform.
