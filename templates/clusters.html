Play Style Clusters

K-Means clustering identifies 12 distinct play style archetypes using PCA dimensionality reduction

About this visualisation

Each dot represents one player-season. The axes (PC1, PC2) are principal components — mathematical summaries of all features compressed into 2 dimensions for visualisation. Clusters were identified using K-Means with k=4. Hover over any dot to see the player name and stats.
